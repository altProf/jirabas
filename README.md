# jirabas

Are you console man?
Then don't spend your time by searching and clicking Jira to find which ticket number you just resolved!

## Installation

    gem install jirabas

## Usage

    jirabas

Will show actual information for you:
-   Tickets in progress;
-   Resolved tickets for past 2 days;
-   Your tickets TODO.
